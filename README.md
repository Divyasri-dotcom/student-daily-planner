# Student Daily Planner

## What the App Does
Student Daily Planner is a simple web application that helps students organize their daily academic work. It helps students view daily tasks, plan study time, and stay organized.

This is the Day 1 version of the project. The goal of Day 1 is to create a clear README specification and a scaffolded app that runs successfully.

## Core Screens

### 1. Home Screen
The Home screen displays the app name and a short introduction.

### 2. Today's Tasks Screen
The Today's Tasks screen displays sample student tasks such as completing assignments, revising subjects, and preparing for class.

### 3. Study Plan Screen
The Study Plan screen shows where the student's daily study plan and revision schedule will be displayed.

### 4. About Screen
The About screen explains the purpose of the app and how it helps students stay organized.
