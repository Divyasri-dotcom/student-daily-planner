# Student Daily Planner

## What the App Does

Student Daily Planner is a simple web application that helps students organize their daily tasks and study plans. It can be used to track assignments, revision work, and daily academic activities.

This is the Day 1 version of the project. The main goal of Day 1 is to create a clear project specification and a runnable app skeleton.

## Core Screens

### 1. Home Screen

The Home screen shows the app name and a short introduction about the purpose of the app.

### 2. Today's Tasks Screen

The Today's Tasks screen shows sample student tasks such as completing assignments, revising subjects, and preparing for class.

### 3. Study Plan Screen

The Study Plan screen explains where the student's daily study schedule and revision plan will be displayed.

### 4. About Screen

The About screen explains why the app is useful for students and how it helps them stay organized.
